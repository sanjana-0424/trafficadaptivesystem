Adaptive Traffic Signal System
AI-based adaptive traffic signal system using Python and OpenCV.

## Features
- Vehicle detection
- Vehicle counting
- Dynamic signal timing
- Real-time traffic analysis

## Technologies Used
- Python
- OpenCV
- Tkinter

## Project Workflow
Traffic Video → Vehicle Detection → Vehicle Counting → Adaptive Signal Timing

## Future Improvements**
